Restaurant Management System
Overview

The Restaurant Management System (RMS) is a comprehensive web application designed to streamline restaurant operations. It offers a seamless experience for customers to order food online, view menus, and provides powerful reporting tools for management.
Features

    Customer Management: Efficiently manage customer information and track their orders.
    Online Ordering: Customers can easily browse the menu, add items to their cart, and place orders online.
    Menu Management: Admins can add, update, and delete menu items, ensuring the menu is always up to date.
    Cart Management: Customers can view and modify their cart before placing orders.
    Feedback System: Customers can provide feedback on their dining experience, which is managed by the admin.
    Reporting Tools: Generate insightful reports to help management make data-driven decisions.

Technologies Used

    Front End: HTML, CSS, JavaScript
    Back End: PHP, MySQL
    Web Server: XAMPP
    Code Editor: Visual Studio Code

Installation

    Clone the repository: git clone https://github.com/yourusername/restaurant-management-system.git
    Set up XAMPP and start Apache and MySQL.
    Import the database: Open phpMyAdmin and import the restaurant_database.sql file located in the database folder.
    Update the database configuration in config.php file.
    Run the application by accessing http://localhost/restaurant-management-system in your web browser.

Usage

    Admin: Access the admin panel to manage customers, feedback entries, and products.
    Customer: Browse the menu, add items to the cart, place orders, and provide feedback.
